# مساعد المحلات والتجار الصغيرين

تطبيق أندرويد للتجار الصغار في مصر يحتوي على 5 ميزات رئيسية:
1. كتابة بوستات تلقائي باللهجة المصرية
2. رد تلقائي على العملاء
3. حساب المكسب
4. تصميم صور إعلانية
5. تذكير بالديون والآجل

## كيفية بناء التطبيق

### طريقة 1: باستخدام Android Studio (الأسهل)
1. قم بتحميل وتثبيت Android Studio من https://developer.android.com/studio
2. افتح Android Studio واختر "Open an Existing Project"
3. اختر مجلد المشروع: `f:\payle\Payle`
4. انتظر حتى Android Studio يقوم بتحميل الاعتماديات (Gradle)
5. بعد انتهاء التحميل، اذهب إلى القائمة: **Build > Build Bundle(s) / APK(s) > Build APK(s)**
6. بعد انتهاء البناء، ستظهر لك رسالة "APK(s) generated successfully"
7. اضغط على زر "locate" في الرسالة لتجذ الملف APK

الملف ستجده في المسار: `app\build\outputs\apk\debug\app-debug.apk`

### تثبيت التطبيق على الموبايل
1. انقل ملف `app-debug.apk` إلى موبايلك
2. من إعدادات الموبايل، قم بتفعيل خيار "السماح بتثبيت التطبيقات من مصادر مجهولة"
3. افتح الملف APK من ملفات الموبايل واضغط على "تثبيت"
4. استمتع بالتطبيق!
